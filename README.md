# Shaoke

A linguistic website built in vue 3 + vite and tailwind

## Packages Installed

- vue 3
- vue-router
- tailwindCSS
- vitejs
- autoprefixer

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
